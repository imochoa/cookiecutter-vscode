# cookiecutter-vscode
Templates for working with Visual Studio Code

# Usage

From https://cookiecutter.readthedocs.io/en/2.1.1/advanced/directories.html

```bash
pipx run \
cookiecutter \
https://github.com/imochoa/cookiecutter-vscode --directory="cpp"
```
